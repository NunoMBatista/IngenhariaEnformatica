Apontamentos das aulas teóricas de CGI de 2025-26.


# Organização de ficheiros 

Ficheiros organizados com extensoes:
- .txt ou .md => apontamentos originais das aulas
- (...)_summary.md => apontamentos revistos e completos à posteriori
- /images => pasta que contem as imagens dos ficheiros .md (se removida, não aparecem as imagens)
